# 📱 Sobre o Projeto

O TelePronto é um aplicativo de atendimento médico remoto criado para ajudar hospitais a desafogar o pronto-socorro físico, permitindo triagem rápida, consultas online e gerenciamento de receitas digitais.

O objetivo principal do projeto é oferecer uma interface simples, clara e acessível, principalmente para usuários idosos ou em situação de mal-estar.

---

# 🖼 Telas Desenvolvidas

O protótipo possui as seguintes telas principais:

1. Home / Dashboard
2. Fluxo de Triagem
3. Sala de Espera Virtual
4. Interface de Videochamada
5. Receita Digital / Minha Farmácia
6. Configuração de Alarmes

---

# 🔄 Fluxo Crítico

O fluxo principal do aplicativo foi pensado para reduzir o número de etapas até o atendimento médico.

### Fluxo:
Home → Consulta Agora → Triagem → Sala de Espera → Videochamada → Receita Digital
